# Help Menu Commands
The **Help** menu provides commands to open Help files.

 **Index**
 Opens the contents screen for Spy++ Help.

 **Using Help**
 Opens a Help file that describes how to use help.

 **About Spy++**
 Shows the software version and copyright information for Spy++.
